Что такое React Native?

React Native — это фреймворк, который позволит вам разрабатывать кроссплатформенные приложения для iOS и Android на JavaScript.

Во время нашего онлайн-курса вы с нуля разработаете приложение на уровне middle-разработчика, что позволит вам в будущем проходить технические собеседования на позицию React Native разработчика.

Востребованы ли React Native разработчики?

Наш ответ – да, разработчики, знающие React Native, востребованы. Многие компании ищут таких специалистов, что означает — они готовы достойно платить.
